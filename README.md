# FarmEdu Ermin 🌾📱

Premium Android aplikacija za edukaciju farmera, poljoprivredu, stočarstvo i organizaciju farmi.
Aplikacija kombinuje moderne Android funkcionalnosti sa Firebase backendom, AI chatom, premium notes sistemom i edukativnim sadržajem za farmere.

# 🚀 Instalacija

## 1. Clone repository

```bash
git clone https://github.com/YOUR_USERNAME/FarmEdu-Ermin.git
```

---

## 2. Open Android Studio

Otvoriti projekat u Android Studio.

---

## 3. Firebase Setup

Dodati:

```text
google-services.json
```

u:

```text
app/google-services.json
```

---

## 4. Enable Firebase Services

U Firebase Console omogućiti:

* Authentication
* Firestore Database
* Firebase Storage

---

## 5. Sync Gradle

```text
File > Sync Project With Gradle Files
```

---

## 6. Run Application

Pokrenuti aplikaciju na emulatoru ili fizičkom uređaju.
